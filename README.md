# Project Lighthouse

Project Lighthouse is a collection of free online calculators, converters, developer utilities and productivity tools.

## Roadmap

- Age Calculator
- Percentage Calculator
- BMI Calculator
- EMI Calculator
- GST Calculator
- SIP Calculator
- Loan Calculator
- Currency Converter
- Unit Converter

Built with:

- Next.js
- React
- TypeScript
